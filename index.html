Quick Tutorial — Chatbots 101

  1. What is a chatbot? A chatbot is a program that interacts with users using natural language.
  2. Types: Rule-based, retrieval-based, generative (large language models).
  3. Basic flow: Input → Intent detection → Response generation → Output
  4. Try it: Follow the minimal example below or build an interactive demo in `presentation/`.
Minimal JS example (client-side) 
// Minimal rule-based example (client-side)
const botResponses = {
  hello: "Hi! I'm a demo chatbot. Ask me about chatbots.",
  chatbot: "Chatbots are programs that can converse. Types include rule-based and AI-based."
};
function respond(message) {
  const m = message.toLowerCase();
  const key = m.match(/\b(hello|chatbot)\b/);
  return key ? botResponses[key[1]] : "Sorry, I don't understand. Try 'hello' or 'chatbot'.";
}
// Usage: respond('hello');
